## THE THREE SURVIVORS — DEEP DIVE

### Survivor 1: **Mediant**

| Aspect | Notes |
|---|---|
| Pronunciation | ME-dee-ant (2 syllables) |
| Meaning | One who mediates. Brokers ARE mediators between buyers and sellers. |
| Voice fit | ✓ Modern, soft ending, real word |
| Multi-segment fit | ✓ Mediating works for builder ↔ contractor, contractor ↔ planner |
| Pros | Real word, semantically perfect, uncommon as brand |
| Cons | Music theory term ("mediant" = third scale degree). Possibly some legal/financial associations. |
| Risk level | Moderate — needs trademark verify |

### Survivor 2: **Vouchdesk**

| Aspect | Notes |
|---|---|
| Pronunciation | VOWCH-desk (2 syllables) |
| Meaning | "Vouch" = to vouch for someone, "desk" = workspace. Brokers vouch for properties. |
| Voice fit | ✓ Modern compound, "desk" feels work-tool-like (reachdesk style) |
| Multi-segment fit | ✓ "Vouching" works across all segments — trust is universal |
| Pros | Compound word feels proprietary, no clear collision found |
| Cons | "Vouch" might feel a bit folksy for premium tech brand |
| Risk level | Lower — but adjacent to ClientTether and other "trust" brand spaces |

### Survivor 3: **Movewell**

| Aspect | Notes |
|---|---|
| Pronunciation | MOOV-well (2 syllables) |
| Meaning | "Move" + "well". Brokers help people MOVE (homes). Doing it WELL. |
| Voice fit | ✓ Real-estate-evocative without "prop" prefix |
| Multi-segment fit | Modest — "move" is broker-centric, less natural for builder workflow |
| Pros | Friendly, evocative, broker-relatable |
| Cons | Might feel limiting as platform expands. "Well" suffix is overused (Matchwell etc.) |
| Risk level | Lower — but Matchwell adjacency, possibly residential moving company collisions I didn't fully check |
